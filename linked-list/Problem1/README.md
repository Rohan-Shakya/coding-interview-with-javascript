# Reverse Linked List

Given a singly linked list, return it in reverse.

**_Example 1:_**

```
Input: 1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL
```

**_Example 2:_**

```
Input: 3
Output: 3
```

## Q/A:

- What do we return if we get null or single node?
  - Just return null and the node back respectively.
